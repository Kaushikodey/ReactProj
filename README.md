# e-plantShopping (Paradise Nursery)

e-plantShopping is a React-based web application for purchasing houseplants online.  
The application allows users to browse plants by category, add items to a shopping cart, and manage cart quantities using Redux.

## Project Name
e-plantShopping

## Description
Paradise Nursery is an online plant shopping platform designed to provide a smooth and user-friendly shopping experience for plant lovers.

## Technologies Used
- React
- Redux Toolkit
- React Router
- CSS

## How to Run the Project
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
